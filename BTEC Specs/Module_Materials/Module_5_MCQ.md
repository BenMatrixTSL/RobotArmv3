# Module 5: Statics, Forces & Load Analysis
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. What is torque?**
   a) Linear force
   b) Rotational force
   c) Speed
   d) Position

**2. Which joint typically carries the most load?**
   a) Elbow
   b) Shoulder
   c) Wrist
   d) Base

**3. What is a working envelope?**
   a) The robot's speed
   b) The safe operating area where the robot can reach
   c) The robot's color
   d) The robot's cost

**4. How do torque limits define safe operation?**
   a) They don't matter
   b) They set maximum safe loads to prevent damage
   c) They make the robot faster
   d) They change the robot's color

**5. How does payload mass affect joint loading?**
   a) It doesn't affect loading
   b) Heavier payloads increase joint loading
   c) Heavier payloads decrease loading
   d) It only affects the wrist

**6. How does reach (arm extension) affect loading?**
   a) It doesn't affect loading
   b) Longer reach increases loading on joints
   c) Longer reach decreases loading
   d) It only affects the base

**7. What happens to stability when the arm is fully extended?**
   a) Stability increases
   b) Stability decreases
   c) Stability stays the same
   d) Stability doesn't matter

**8. Why do different postures create different loads?**
   a) They don't
   b) Because leverage and moment arms change
   c) Because the robot changes color
   d) Because the speed changes

**9. How can you use load readings to stay within safe limits?**
   a) Ignore them
   b) Monitor them and stop if approaching limits
   c) Increase speed
   d) They're not useful

**10. What is the relationship between reach and payload?**
   a) No relationship
   b) Longer reach reduces maximum safe payload
   c) Longer reach increases maximum safe payload
   d) They're the same thing

---

## Answer Key

1. **b)** Rotational force
2. **b)** Shoulder
3. **b)** The safe operating area where the robot can reach
4. **b)** They set maximum safe loads to prevent damage
5. **b)** Heavier payloads increase joint loading
6. **b)** Longer reach increases loading on joints
7. **b)** Stability decreases
8. **b)** Because leverage and moment arms change
9. **b)** Monitor them and stop if approaching limits
10. **b)** Longer reach reduces maximum safe payload
