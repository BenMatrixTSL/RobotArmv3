# Module 10: Data Logging & Diagnostics
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. Why is data logging important?**
   a) It's not important
   b) To diagnose problems and optimize performance
   c) To make the robot faster
   d) To change the robot's color

**2. What data should be logged?**
   a) Only position
   b) Joint angles, loads, voltages, errors
   c) Only speed
   d) Only temperature

**3. Why is timestamping important?**
   a) It's not important
   b) To correlate events and understand timing
   c) To make logs smaller
   d) To make logs bigger

**4. What is a serial timeout?**
   a) When communication takes too long
   b) When the robot moves too fast
   c) When the robot stops
   d) When data is invalid

**5. What causes invalid position data?**
   a) Communication errors or sensor issues
   b) The robot moving
   c) The robot stopping
   d) Normal operation

**6. What indicates a power/voltage dip?**
   a) Sudden drop in voltage readings
   b) Increase in voltage
   c) Constant voltage
   d) No voltage

**7. What indicates an overload condition?**
   a) High load readings
   b) Low load readings
   c) No load
   d) Constant load

**8. How can logs help optimize polling intervals?**
   a) They can't help
   b) By showing response times and system behavior
   c) By making it faster
   d) By making it slower

**9. How can logs help optimize timeouts?**
   a) They can't help
   b) By showing typical response times
   c) By making timeouts longer
   d) By removing timeouts

**10. What is the purpose of diagnostic analysis?**
   a) To waste time
   b) To identify problems and improve reliability
   c) To make the robot faster
   d) To change settings randomly

---

## Answer Key

1. **b)** To diagnose problems and optimize performance
2. **b)** Joint angles, loads, voltages, errors
3. **b)** To correlate events and understand timing
4. **a)** When communication takes too long
5. **a)** Communication errors or sensor issues
6. **a)** Sudden drop in voltage readings
7. **a)** High load readings
8. **b)** By showing response times and system behavior
9. **b)** By showing typical response times
10. **b)** To identify problems and improve reliability
