# Module 9: Software Architecture & Advanced Programming
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. What is the Electron front-end responsible for?**
   a) Low-level servo control
   b) User interface and visualization
   c) Power supply
   d) Mechanical structure

**2. What does the Node.js/WebSocket server do?**
   a) Provides power
   b) Handles communication between front-end and servos
   c) Moves the robot
   d) Holds the robot together

**3. What is joint-space control?**
   a) Commanding XYZ positions
   b) Commanding joint angles
   c) Commanding speed only
   d) Commanding load

**4. What is Cartesian-space control?**
   a) Commanding joint angles
   b) Commanding XYZ positions
   c) Commanding speed only
   d) Commanding load

**5. What is a motion profile?**
   a) How the robot moves over time (speed/acceleration)
   b) The robot's appearance
   c) The robot's cost
   d) The robot's weight

**6. How does high acceleration affect smoothness?**
   a) Improves smoothness
   b) Reduces smoothness (more jerky)
   c) No effect
   d) Makes it slower

**7. How does high speed affect mechanical stress?**
   a) Reduces stress
   b) Increases stress
   c) No effect
   d) Makes it smoother

**8. What is the trade-off between speed and accuracy?**
   a) Higher speed typically reduces accuracy
   b) Higher speed increases accuracy
   c) No relationship
   d) They're the same

**9. What is a coordinated multi-joint move?**
   a) Moving joints independently
   b) Moving multiple joints together with shared timing
   c) Moving one joint at a time
   d) Stopping all joints

**10. Why is understanding software architecture important?**
   a) It's not important
   b) It helps you modify and extend the system
   c) It makes the robot faster
   d) It changes the robot's color

---

## Answer Key

1. **b)** User interface and visualization
2. **b)** Handles communication between front-end and servos
3. **b)** Commanding joint angles
4. **b)** Commanding XYZ positions
5. **a)** How the robot moves over time (speed/acceleration)
6. **b)** Reduces smoothness (more jerky)
7. **b)** Increases stress
8. **a)** Higher speed typically reduces accuracy
9. **b)** Moving multiple joints together with shared timing
10. **b)** It helps you modify and extend the system
