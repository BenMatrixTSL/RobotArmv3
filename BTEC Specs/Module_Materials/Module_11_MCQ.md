# Module 11: Mechatronic System Integration
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. What is mechatronics?**
   a) Only mechanics
   b) Integration of mechanics, electronics, and software
   c) Only electronics
   d) Only software

**2. What are actuators in a robot arm?**
   a) Sensors
   b) Motors/servos that move the arm
   c) The controller
   d) The base

**3. What are sensors in a robot arm?**
   a) Components that provide feedback
   b) The motors
   c) The controller
   d) The base

**4. What is accuracy?**
   a) How close the robot gets to the target position
   b) How fast it moves
   c) How much it weighs
   d) How much it costs

**5. What is repeatability?**
   a) How consistently the robot returns to the same position
   b) How fast it moves
   c) How much it weighs
   d) How much it costs

**6. What is backlash?**
   a) Play or gap in joints causing inaccuracy
   b) The robot moving backward
   c) The robot stopping
   d) The robot moving fast

**7. How does backlash affect accuracy?**
   a) It doesn't affect it
   b) It reduces accuracy
   c) It improves accuracy
   d) It makes it faster

**8. How does mechanical design affect accuracy?**
   a) It doesn't affect it
   b) Backlash and stiffness influence accuracy
   c) It only affects speed
   d) It only affects cost

**9. How does control strategy affect repeatability?**
   a) It doesn't affect it
   b) Control algorithms influence how consistently the robot moves
   c) It only affects speed
   d) It only affects appearance

**10. Why is system integration important?**
   a) It's not important
   b) All components must work together for optimal performance
   c) It makes the robot faster
   d) It changes the robot's color

---

## Answer Key

1. **b)** Integration of mechanics, electronics, and software
2. **b)** Motors/servos that move the arm
3. **a)** Components that provide feedback
4. **a)** How close the robot gets to the target position
5. **a)** How consistently the robot returns to the same position
6. **a)** Play or gap in joints causing inaccuracy
7. **b)** It reduces accuracy
8. **b)** Backlash and stiffness influence accuracy
9. **b)** Control algorithms influence how consistently the robot moves
10. **b)** All components must work together for optimal performance
