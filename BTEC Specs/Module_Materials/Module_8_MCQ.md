# Module 8: Advanced Control & Communication
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. What is serial communication?**
   a) Parallel data transmission
   b) Sequential data transmission one bit at a time
   c) Wireless communication
   d) No communication

**2. What is baud rate?**
   a) The speed of data transmission
   b) The robot's speed
   c) The robot's position
   d) The robot's weight

**3. What is a checksum used for?**
   a) Error detection
   b) Speed control
   c) Position control
   d) Power supply

**4. What contributes to system latency?**
   a) Serial speed, polling, timeouts, queueing
   b) The robot's color
   c) The robot's size
   d) The robot's cost

**5. What happens if a timeout is too short?**
   a) Commands may be cancelled prematurely
   b) Nothing happens
   c) The robot moves faster
   d) The robot stops

**6. What happens if a timeout is too long?**
   a) System may wait unnecessarily
   b) Nothing happens
   c) The robot moves faster
   d) The robot stops

**7. What does the "moving flag" indicate?**
   a) Whether the servo is currently moving
   b) The servo's position
   c) The servo's speed setting
   d) The servo's temperature

**8. What does the "torque flag" indicate?**
   a) Whether torque is enabled or disabled
   b) The servo's speed
   c) The servo's position
   d) The servo's temperature

**9. How can flags help with safety?**
   a) By indicating servo state and enabling safety checks
   b) They don't help
   c) By making the robot faster
   d) By changing the robot's color

**10. What is status polling?**
   a) Regularly checking servo status
   b) Stopping the robot
   c) Moving the robot
   d) Programming the robot

---

## Answer Key

1. **b)** Sequential data transmission one bit at a time
2. **a)** The speed of data transmission
3. **a)** Error detection
4. **a)** Serial speed, polling, timeouts, queueing
5. **a)** Commands may be cancelled prematurely
6. **a)** System may wait unnecessarily
7. **a)** Whether the servo is currently moving
8. **a)** Whether torque is enabled or disabled
9. **a)** By indicating servo state and enabling safety checks
10. **a)** Regularly checking servo status
