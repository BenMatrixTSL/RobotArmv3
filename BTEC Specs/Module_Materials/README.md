# Module Materials Index

This folder contains worksheets, assessment rubrics, and multiple choice questions for all 16 modules of the BTEC Robot Arm Curriculum.

## Structure

Each module has three files:
- `Module_X_Worksheet.md` - Student worksheet with exercises and questions
- `Module_X_Rubric.md` - Assessment rubric for marking
- `Module_X_MCQ.md` - Multiple choice questions for testing

- All materials are beginner-friendly and aligned with BTEC learning objectives
