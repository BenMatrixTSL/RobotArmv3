# Module 6: Industrial Context & Basic Automation
## Multiple Choice Questions

**Instructions:** Choose the best answer for each question.

---

**1. Which is NOT a common industrial robot application?**
   a) Pick-and-place
   b) Assembly
   c) Cooking meals
   d) Welding

**2. What is "repeatability" in robotics?**
   a) How fast the robot moves
   b) How accurately the robot returns to the same position
   c) How much weight it can lift
   d) How far it can reach

**3. What is a "waypoint" in a sequence?**
   a) A stopping point
   b) A taught position in a sequence
   c) The end effector
   d) The base

**4. What is a "dwell" time?**
   a) A pause in the sequence
   b) Moving faster
   c) Stopping permanently
   d) The robot's speed

**5. What is a "home" position?**
   a) A safe starting/return position
   b) The robot's base
   c) The end effector
   d) The controller

**6. Why is repeatability important?**
   a) It makes the robot look better
   b) It ensures consistent operation in automated processes
   c) It makes the robot faster
   d) It's not important

**7. What is "cycle time"?**
   a) The time for one complete sequence
   b) The robot's age
   c) The number of joints
   d) The robot's speed

**8. How do sequences fit into larger processes?**
   a) They don't
   b) They are part of automated workflows
   c) They are standalone
   d) They only work alone

**9. What is the main difference between educational and industrial robots?**
   a) Color
   b) Industrial robots typically have higher payload, speed, and accuracy
   c) Size only
   d) There's no difference

**10. Why test repeatability multiple times?**
   a) To waste time
   b) To verify consistency and reliability
   c) To make it faster
   d) It's not necessary

---

## Answer Key

1. **c)** Cooking meals
2. **b)** How accurately the robot returns to the same position
3. **b)** A taught position in a sequence
4. **a)** A pause in the sequence
5. **a)** A safe starting/return position
6. **b)** It ensures consistent operation in automated processes
7. **a)** The time for one complete sequence
8. **b)** They are part of automated workflows
9. **b)** Industrial robots typically have higher payload, speed, and accuracy
10. **b)** To verify consistency and reliability
